# Sixty-Below

## Gameplay

* Monde généré procéduralement de 2048 par 768 tuiles de 16 pixels de côté
* Vue de côté
* Exploration : déplacement par flèches directionnelles, saut, nage, plongée, Monde varié (surface, tunnels, cavernes, mer, forêt, jungle, désert, lacs)
* Minéral : minage (substrat, minerai, gemmes...), construction
* Flore : récolte dans le monde (arbres, champignons, herbes, fleurs, coraux), plantation
* Faune : Combat au tour par tour (Arène générée procéduralement, Points d'Action / Points de Mouvement, Buff Panoplie)
* Artisanat : Tout ce qui ne se loote pas se crafte (Ingrédients, Recettes, Crafting Station)
* Housing : Shelter (protection nocturne), Crafting Station, meubles
* Liquides : Simulation d'écoulement, Pêche (poissons, appâts)
* Buffs : statiques, dynamiques, nourriture, potion, armures, outils, accessoires
* Stockage : Inventaire, Coffres
* Aide en ligne
* Succès - Mesure de l'avancement dans le jeu
